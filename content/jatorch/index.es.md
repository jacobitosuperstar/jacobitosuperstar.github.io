## **JaTorchC - Recreación de PyTorch en C**

🔗 [Ver en Codeberg](https://codeberg.org/jacobitosuperstar/JaTorchC)

Un proyecto de aprendizaje para entender profundamente los fundamentos del machine learning recreando PyTorch desde cero en C, con planes para una interfaz en Python.

### **Visión del Proyecto**

JaTorchC busca proporcionar una comprensión práctica de:
- Cómo funciona la diferenciación automática internamente
- Operaciones con tensores y gestión de memoria
- Bloques de construcción de redes neuronales desde primeros principios
- El puente entre C de bajo nivel e interfaces de alto nivel en Python

### **¿Por Qué Este Proyecto?**

Construir un framework de deep learning desde cero es una de las mejores formas de entender verdaderamente:
- Cómo fluyen los gradientes a través de grafos computacionales
- Disposición de memoria y optimización para operaciones con tensores
- Las decisiones de diseño detrás de frameworks populares como PyTorch
